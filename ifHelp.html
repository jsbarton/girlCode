IF STATEMENTS


Congratulations on making it to this coding challenge! An if statement is a type of conditional that executes a block of code when the statement is true. So, whatever comes after the brackets in an if statement will on happen if the statment is true. For example, let's try a sentence: 

 if  I do my homework, I will get a good grade.
Code for this would look like: 
   if (I do my homework){
    I will get a good grade.
  }
